# Perfect Markdown

**The most complete Markdown widget for Figma — every syntax, beautifully rendered.**

---

## ✨ Why Perfect Markdown?

Most Markdown tools for Figma fall short: tables render as raw text, formatting is limited, and customization is an afterthought. **Perfect Markdown** handles virtually every Markdown syntax and renders it as a polished, visual document right on your canvas.

---

## 🎯 Key Features

### 📊 Visual Tables
Other tools display tables as `|---|---|` — we render **actual visual tables** with:
- Styled header rows with background color
- Alternating row colors for readability
- Proper cell borders and padding
- Automatic text wrapping in cells
- Row height alignment (all cells in a row match the tallest)
- Full inline formatting inside cells (bold, italic, code, links, strikethrough)

### 🎨 Color Swatches
Write a hex color code like `#1a4cb3` anywhere — in tables or text — and it automatically displays with a **visual color swatch**. Perfect for design token documentation.

### ⚡ Instant Width Toggle
Switch between 4 preset widths with a **single click**:
- **S** (400px) — Compact notes
- **M** (600px) — Standard documents
- **L** (800px) — Wide content
- **XL** (1200px) — Full-width specs

The `↔` button in the toolbar cycles through sizes instantly — no menu diving required.

### 🌗 Light & Dark Themes
Seamlessly switch between themes to match your design context. Both themes feature carefully tuned contrast ratios for optimal readability.

### ✅ Checkboxes & Task Lists
Create task lists with `- [ ]` and `- [x]` syntax. Checked items display with strikethrough for clear progress tracking.

### 🔗 Clickable Links
Links written as `[text](url)` are **actually clickable** — opens the URL when clicked.

### 📝 Full Markdown Syntax
- **Headings** (H1-H6) with proper hierarchy
- **Bold**, *italic*, and ~~strikethrough~~ formatting
- Bullet and numbered lists with **nesting support**
- `Inline code` and code blocks with monospace styling
- Horizontal rules
- Color swatches for hex codes
- Visual tables with full cell formatting

## 🚀 How to Use

1. **Add the widget** to your canvas
2. **Click "Edit Markdown"** in the toolbar to write your Markdown
3. **Click `↔`** to cycle through widths
4. **Switch themes** via the toolbar dropdown
